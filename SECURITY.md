# Security Policy

## Supported versions

This project is currently in an early release stage. The `main` branch is considered the primary supported version.

Security fixes will generally be applied to:

- The latest tagged release, and
- The `main` branch.

## Reporting a vulnerability

If you believe you have found a security vulnerability in this project:

1. Do **not** open a public GitHub issue with sensitive details.
2. Instead, contact the maintainers privately using one of the following:
   - GitHub security advisories, or
   - Direct contact information listed on the maintainer's GitHub profile or associated organization.
3. Provide:
   - A description of the issue
   - Steps to reproduce
   - Any relevant logs or configuration details (redacted of secrets)

We aim to acknowledge reports promptly and will work with you to validate the issue, determine impact, and release a fix where appropriate.
